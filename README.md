# Project Thông tin Sinh viên

## Cách chạy:

### 1. HTML (index.html)
- Mở trực tiếp file `index.html` trong trình duyệt

### 2. PHP (students.php)  
- Cần XAMPP/WAMP hoặc PHP server
- Chạy: `php -S localhost:8000`
- Truy cập: http://localhost:8000/students.php

### 3. Node.js (server.js)
- Chạy: `node server.js`
- Truy cập: http://localhost:3000

## Thông tin hiển thị:
- Tên sinh viên
- Mã sinh viên  
- Lớp
- Môn học
